01 - Le réveil

Vous ouvrez les yeux, la tête lourde, accablé(e) par une douleur vive. Vous regardez autour, sans trop comprendre comment vous avez pu vous retrouver dans cette pièce.

« Mais qu'est-ce que ... »

Un texte inscrit sur le mur attire votre attention, mais pour le lire, il vous faut vous approcher.
